# GeoTools

GeoTools 是一个有用的地理工具应用程序，旨在帮助用户进行各种地理相关的任务。

## 功能

- 地理位置显示
- 路线规划
- 地图标记
- 实时天气信息

## 安装

请按照以下步骤安装和运行此项目：

1. 克隆仓库：

   ```sh
   git clone https://github.com/littlepenguin66/GeoTool.git
   cd geotools
   ```

2. 安装依赖：

   ```sh
   flutter pub get
   ```

3. 运行应用：
   ```sh
   flutter run
   ```
